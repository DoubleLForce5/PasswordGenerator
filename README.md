# Secure Password Generator

## Description

This application allows users to generate a random password based on selected criteria.

## Usage instructions
On page load, the user is presented with the password generator.  

<img width="837" alt="Screen Shot 2021-06-21 at 8 33 16 AM" src="https://user-images.githubusercontent.com/73543476/122770705-55568800-d26b-11eb-8d39-28b1bc8d6016.png">

Upon clicking the 'generate password' button the user is prompted to pick a number between 8 and 128 for the password's length. 

<img width="448" alt="Screen Shot 2021-06-21 at 8 36 24 AM" src="https://user-images.githubusercontent.com/73543476/122771141-c5650e00-d26b-11eb-9925-54f4555491d4.png">

If the user selects a password length outside of the required range then the application will inform the user that their selection is invalid. 

<img width="444" alt="Screen Shot 2021-06-21 at 8 40 12 AM" src="https://user-images.githubusercontent.com/73543476/122771720-4cb28180-d26c-11eb-9539-0f479eee6376.png">

Once an appropriate password length has been selected the user will receive a series of prompts asking if they would like numbers, special characters, uppercase letters, and lowercase letters included in their password. 

<img width="446" alt="Screen Shot 2021-06-21 at 8 43 12 AM" src="https://user-images.githubusercontent.com/73543476/122772261-cea2aa80-d26c-11eb-8cb2-654c9772d6bd.png">
<img width="445" alt="Screen Shot 2021-06-21 at 8 43 22 AM" src="https://user-images.githubusercontent.com/73543476/122772274-d3675e80-d26c-11eb-95f1-6cc1b397b173.png">
<img width="447" alt="Screen Shot 2021-06-21 at 8 43 32 AM" src="https://user-images.githubusercontent.com/73543476/122772276-d5312200-d26c-11eb-9a0f-460f0e482647.png">
<img width="443" alt="Screen Shot 2021-06-21 at 8 43 42 AM" src="https://user-images.githubusercontent.com/73543476/122772282-d6fae580-d26c-11eb-9ed6-8e24e0ace8d8.png">

Following the prompts, a password will be generated based on the user's criteria. 
The user can then decide to keep the password or generate a new one based on different criteria. 

<img width="718" alt="Screen Shot 2021-06-21 at 8 47 46 AM" src="https://user-images.githubusercontent.com/73543476/122772821-5f798600-d26d-11eb-9eba-f96c2106e161.png">
